# Как запустить проект локально

## Для запуска приложения на локальном компьютере:

1. Перейдите в консоли в папку проекта.

2. Выполните команду `dotnet run`.

3. Дождитесь запуска приложения.

4. Перейдите в браузере по адресу `http://localhost:5001/`

5. Откроется простая веб-страница для тестирования endpoint приложения.

6. Нажмите кнопку "Залогиниться".

7. После этого можно работать со списком дел через кнопки "Показать список дел", "Добавить дело", "Показать дело", "Изменить дело", "Удалить дело".

8. Чтобы показать, изменить или удалить дело, нужно указать его номер. Чтобы добавить или изменит дело, нужно заполнить поля Title и IsCompleted.

9. Для проверки работы приложения с БД можно использовать программу для работы с БД (например DB Browser for SQLite).

10. Фронтенд использован потому, что доступ к endpoint через адресную строку невозможен из-за необходимости авторизации: в адесной строке нужно каждый раз при смене адреса передавать токен. Во фронтенде передача токена происходит автоматически.
